MINIMA is a pragmatic and lightweight innovation model designed for public administrations.
It provides a minimal yet structured approach to initiate innovation, while allowing
progressive evolution as organizational maturity increases.

The model focuses on action, learning, and adaptability rather than exhaustive frameworks
or heavy governance.
